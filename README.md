# Durandal + Webpack

This is a sample project that demonstrates the use of Webpack with Durandal together to bundle your SPA.

## Quickstart

Install the package dependencies with
```
npm install
```

Start the webpack at `localhost:8080` with
```
npm run dev
```

Go forth!
